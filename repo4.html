Repositório:rinhabackend-rails-api

Descrição:

This project was inspired by a community driven challenge that ran on August 2023: Challenge INSTRUCTIONS: Challenge STRESS TEST Unfortunatelly I only heard about it a few days after it was closed, but I decided to try it out here. The API itself is not the challenge, but to make it fit within the confines of a measle 1.5 vCPU and 3GB of RAM, and endure a very heavy, almost DDoS-like Gatling stress test. It's brutal test. The goal of this version was to make a fully Rails API based app, with enough speed optimizations without completely breaking the framework. Some tricks are not recommended for real production usage, but it leverages the fact that this is a performance oriented challenge.
